# uart-atmega328p
Part of uart-avr, application for printing ADC results

Инструкция по настройке драйвера:

1. Установить компилятор и утилиты для работы с AVR
$> sudo pacman -S avr-gcc avr-libc avrdude

2. Проверить, что драйвер для преобразователя USB-RS232 установлен
$> lsusb
В списке должно быть устройство CH340 или HL340 или что-то похожее
Если его нет, установить драйвер из файла CH340.zip по инструкции внутри.

3. Чтобы собрать программу из исходного когда, переключитесь в папку с этим файлом README и наберите команду
$> make clean
$> make all

После этого в папке должны появиться файлы .elf и .hex

4. После этого подключите плату к ЭВМ и найдите в папке /dev/ это устройство. Скорее всего, это будет /dev/serial/by-id/usb...., но это зависит от сборки GNU/Linux. Этот путь вбейте в файл makefile в строчке avrdude... Чтобы записать hex-файл в МК, наберите 
$> make prog
Программа avrdude должна сообщить об успешности записи
